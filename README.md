# 🚀 PORTFOLIO — Ayush | Creative Developer & Designer

A modern, high-performance personal portfolio website built with **React**, **Vite**, **Spline 3D**, and **Lenis**. Featuring an interactive 3D robot hero, premium glassmorphism interfaces, fluid custom scroll physics, dynamic 12-point cursor trails, and stabilized responsive layouts.

Live Demo: [portfolio-beta-bice-sqyyghflsz.vercel.app](https://portfolio-beta-bice-sqyyghflsz.vercel.app/)

---

## ✨ Features & Enhancements

- 🤖 **Interactive 3D Hero** — Animated Spline robot with real-time viewport adjustments and responsive centering.
- 🌊 **ReactLenis Smooth Scroll** — Custom scroll physics via `<ReactLenis>` wrapper, configured with document-variable synchronization and touch scaling normalization.
- 💫 **12-Point Cursor Trail** — Fluid multi-dot SVG mouse tracker following the user's cursor with custom spring-damping physics.
- 🎨 **Premium UI/UX** — Glassmorphic navbar styled to float permanently, gradient background orbs, micro-animations, and theme variables.
- 📱 **Mobile Responsive & Stabilized** — Content boundaries secured on mobile screens to prevent horizontal overflow, with CPU-heavy particles deactivated on smaller devices to prevent layout flickering.
- ✉️ **Interactive Contact Form** — Fully responsive contact section utilizing the Web3Forms API with live sending states.

## 🚀 Getting Started

Follow these steps to run the portfolio locally on your machine:

```bash
# 1. Clone the repository
git clone https://github.com/AAYUSH-KANSAL/PORTFOLIO.git

# 2. Navigate to the project directory
cd PORTFOLIO

# 3. Install dependencies
npm install

# 4. Start the local development server
npm run dev -- --host

# 5. Build production bundle
npm run build
```
