# PORTFOLIO — Ayush | Creative Developer & Designer

A modern, high-performance personal portfolio website built with **React**, **Vite**, and **Spline 3D**. Featuring an interactive 3D robot character, glassmorphism UI, smooth scroll animations, and a fully responsive design.

## ✨ Features

- 🤖 **Interactive 3D Hero** — Animated Spline robot with real-time hand-wave animation
- 🎨 **Premium UI** — Glassmorphism navbar, gradient orbs, micro-animations
- 📱 **Fully Responsive** — Optimized for desktop, tablet, and mobile
- ⚡ **Lightning Fast** — Built with Vite for instant HMR and optimized builds
- 🧩 **Modular Design** — Clean component architecture with React

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| React | UI Components |
| Vite | Build Tool & Dev Server |
| Spline | 3D Interactive Models |
| CSS3 | Styling & Animations |
| JavaScript | Logic & Interactivity |

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## 📂 Project Structure

```
├── App.jsx          # Main application component

├── index.html       # HTML template
├── style.css        # Global styles & design system
├── vite.config.js   # Vite configuration
└── dist/            # Production build output
```
